# Big Data Exercises (Hadoop & Spark)

This repository contains a collection of exercises and solutions for Big Data processing using Apache Hadoop (MapReduce) and Apache Spark in Java.

## Hadoop Exercises

### **Ex_1 & Ex_5: Filter Users by Pricing Plan**
Identifies countries where 100% of the users are on a "free" pricing plan and counts those users.

### **Ex_3 & Ex_4: Multi-Stage Workflow Templates**
Implementation patterns and skeletons for executing multi-stage and two-step MapReduce jobs.

### **Ex_6: Energy Efficiency Analysis**
Finds the most frequent energy efficiency class for buildings grouped by city and construction year.

## Spark Exercises

### **Ex_8: Product Sales Trends**
Analyzes annual sales data to find products with declining trends and identifies top-selling products per year.

### **Ex_9: Word Frequency Filtering**
Advanced text processing to filter lines by prefix and identify high-frequency words based on a dynamic threshold.

### **Ex_10: Station Status & Criticality Analysis**
Evaluates time-series status logs to identify the most critical timeslot (highest rate of zero free slots) for each station.

## Technologies Used
- Apache Hadoop (MapReduce)
- Apache Spark
- Java
